# 是男人就下100层

经典 Flash 游戏「是男人就下100层」的 HTML5 复刻版。纯前端实现，无需任何框架或构建工具，打开即玩。

**在线试玩：https://blog.feuvan.net/man100/**

## 玩法

控制角色不断向下跳落平台，躲避天花板尖刺，挑战最深层数。

**操作方式：**
- 键盘：`← →` 方向键 或 `A / D` 键移动
- 移动端：屏幕底部虚拟方向按钮

## 平台类型

| 颜色 | 类型 | 说明 |
|------|------|------|
| 🟢 绿色 | 安全平台 | 正常站立 |
| 🔵 蓝色 | 移动平台 | 会左右移动，站上去会跟着走 |
| 🟡 黄色 | 易碎平台 | 踩上后短暂停留即碎裂 |
| 🔴 红色 | 尖刺平台 | 踩上即死 |

## 死亡条件

- 被天花板尖刺顶到
- 掉出屏幕下方
- 踩到红色尖刺平台

## 特性

- 纯 HTML5 Canvas + CSS + JavaScript，零依赖
- 像素风角色 + 行走动画
- 难度随层数递增（滚动加速、危险平台比例增加）
- 粒子特效（平台碎裂、角色死亡）
- 本地存储最高纪录
- 响应式适配：移动端全屏 + 桌面端等比缩放
- 移动端虚拟方向按钮

## 快速开始

直接用浏览器打开 `index.html` 即可，或使用任意静态服务器：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

## 项目结构

```
index.html   — 页面结构（开始/结束画面、Canvas、虚拟按键）
style.css    — 样式（UI 覆盖层、响应式、触控按钮）
game.v3.js   — 游戏引擎（物理、碰撞、渲染、难度系统）
```
